Просто кнопка. Чтобы создать кнопку без определенного действия указываем button. Новые типы полей С появлением стандарта HTML5 добавились и новые типы полей форм. В первую очередь они стали гораздо удобнее для пользователей. Ввод E-Mail, type="email". Текстовое поле, у которого на клавиатуре мобильных устройств появляется символ @. Номер телефона, type="tel". На мобильных устройствах открывается клавиатура с числами. Ввод ссылки, type="url". Числовое поле, type="number". Помимо клавиатуры с цифрами появляется возможность переключать значения поля. Атрибуты min и max определяют нижнее и верхнее возможное значение, step - шаг изменения, a value - начальное значение. Числовой ползунок, type="range". Появляются уже указанные атрибуты min, max, step и ﻿value. Поиск, type="search". Google Chrome добавляет крестик для очистки введенной строки. На мобильных устройствах появляется кнопка поиска.
